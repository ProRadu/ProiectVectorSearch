## 5. Exemple de rulare a fișierului cautare.py <a name="Rulare"></a>
Se observă rezultate subunitare deoarece este calculată distanța cosinus ca și diferență de 1-similaritatea cosinus(calculată de oracle ca produs cartezian între 2 vectori, totul împărțit la produsul normelor celor 2 vectori)<br>
Astfel, cu cât rezultatul este mai aproape de 0, cu atât rezultatele sunt mai apropiate.<br>
0 ar însemna texte identice, iar 1 ar însemna nicio similaritate.<br>
Este utilă distanța cosinus pentru că permite să observăm diferențele unghiurilor din spațiul vectorial.<br>
Dacă am fi folosit distanța euclidiană, răspunsuri care s-ar fi aflat în același unghi față de întrebarea pusă de utilizator ar fi putut avea disțante diferite datorită lungimii textelor.<br>

## 6. Ghid de reproducere a proiectului <a name="Ghid"></a>
Pentru ca acest proiect să fie rulat pe un alt sistem, trebuie parcurși următorii pași:

### Pasul 1: Configurare Oracle Cloud (OCI)
1.  **Instanță DB:** Creați o bază de date **Autonomous 26ai** (Always Free) în consola Oracle Cloud.
2.  **Acces Rețea:** În pagina bazei de date, mergeți la **Network** -> **Access Control List (ACL)** și adăugați IP-ul public al calculatorului(butonul Add My IP Address).
3.  **Securitate TLS:** Mergi la **Mutual TLS (mTLS)** și apasă pe **Edit**. **Debifează** căsuța "Require Mutual TLS (mTLS) authentication". Acest pas este esențial pentru a permite conexiunea "Thin" din Python fără fișiere de tip Wallet.

### Pasul 2: Pregătirea Mediului Local
Clonați repository-ul și instalați librăriile necesare:
în consola mediului de dezvoltare pentru proiect, rulați următoarea comandă
<pre>
pip install -r requirements.txt
</pre>

### Pasul 3: Configurarea fișierului .env

Pentru a securiza datele de acces, proiectul utilizează variabile de mediu. Urmați acești pași:

1. Redenumiți fișierul `.env.example` în `.env`.
2. Deschideți fișierul `.env` și completați următoarele câmpuri:
   * **user_db** = Numele utilizatorului din baza de date (ex: `ADMIN`).
   * **parola_db** = Parola de conectare stabilită la crearea instanței Oracle.
   * **dsn_db** = Adresa de conectare (Connection String) completă. 
Puteți obține **dsn_db** din consola Oracle Cloud -> Database Connection -> secțiunea TLS (alegeți șirul de text lung care începe cu `(description=...`).*

### Pasul 4: Execuția Programului
Populare: Rulați <pre>python incarcaVectori.py</pre> pentru a citi CSV-ul, a genera vectorii și a-i salva în Cloud.

Căutare: Rulați <pre>python cautare.py</pre> pentru a interoga baza de date folosind limbaj natural.
